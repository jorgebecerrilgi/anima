## About The Project

<img src="https://user-images.githubusercontent.com/110145796/188253700-a254d8aa-17ef-4a00-b4ce-702d8a1d16eb.gif" width="40%" />

An interactive website to display and control your spritesheet animations.

### Built With
* HTML
* CSS
* Javascript
* JQuery
* Bootstrap

### Upcoming Features
* Import and export your projects!
* Delete animations and states.
* New transition options.
  * On animation end.
  * On timer ends.
  * Flip image.
* New animation options.
  * Customize frames' order.

<!-- USAGE EXAMPLES -->
## Usage

### Settings

Go to the Settings tab to customize the spritesheet.
* Upload your own image.
* Set the frames' size.

<img src="https://user-images.githubusercontent.com/110145796/188254044-c57b52ae-e418-430d-8d9b-f97a60633e5b.gif" width="40%" />

### Animations

Go to the Animations tab to customize the spritesheet animations.
* Create new animations with the 'New Animation' button (duh!).
* Edit current animations by clicking on them.

<img src="https://user-images.githubusercontent.com/110145796/188254725-3233df69-3d92-430b-8b5a-8066de7a3bbb.gif" width="40%" />

### State Machine

Go to the State Machine tab to customize the animation's behaviour.
* Change the starting animation with the Configuration button.
* Enter an animation's transition list by clicking on them.
* Add new transitions.
* Edit current transitions.

<img src="https://user-images.githubusercontent.com/110145796/188254952-569d9507-6053-4a70-8e63-591600320b35.gif" width="40%" />

### Play

Go to the Play tab to visualize and control your spritesheet animations!

<img src="https://user-images.githubusercontent.com/110145796/188253700-a254d8aa-17ef-4a00-b4ce-702d8a1d16eb.gif" width="40%" />
